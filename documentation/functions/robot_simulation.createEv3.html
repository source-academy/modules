    Function createEv3

    • createEv3(physics: Physics, renderer: Renderer): DefaultEv3

      Create an EV3.

      The resulting EV3 should be saved to the context using saveToContext.

      The returned EV3 object is designed to be added to the world using addControllerToWorld.

      This is a Controller function and should be called within init_simulation.

      Parameters

      • physics: Physics

        The physics engine of the world. See createPhysics

      • renderer: Renderer

        The renderer engine of the world. See createRenderer

      Returns DefaultEv3

      EV3

      Example

      init_simulation(() => {
        ...
        const ev3 = createEv3(physics, renderer);
        saveToContext('ev3', ev3);
      })
