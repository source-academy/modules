          render: Renderer,

      ): Paper

      Creates a paper on the floor.

      The returned Paper object is designed to be added to the world using addControllerToWorld.

      This is a Controller function and should be called within init_simulation.

      Parameters

      • render: Renderer

        The renderer engine of the world. See createRenderer

      • url: string

        The url of the image to be displayed on the paper.

      • width: number

        The width of the paper in meters.

      • height: number

        The height of the paper in meters.

      • x: number

        The x position of the paper.

      • y: number

        The y position of the paper.

      • rotation: number

        The rotation of the paper in degrees.

      Returns Paper
