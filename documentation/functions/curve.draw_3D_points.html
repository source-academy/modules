Function draw_3D_points

  • draw_3D_points(numPoints: number): RenderFunction

  • Returns a function that turns a given 3D Curve into a Drawing, by sampling the 3D Curve at num sample points. The Drawing consists of isolated points, and does not connect them. The parts between (0,0,0) and (1,1,1) of the resulting Drawing are shown within the unit cube.
